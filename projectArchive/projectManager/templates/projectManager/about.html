Welcome to my second attempt at creating a usable project archive for Web Team. My first attempt lived in OneDrive. I tried my best to replicate a relational database in my environment, but at the end of the day it's a file system not a database.

This second iteration began as a semester-long project for my web development course. Since I have touched thousands of webpages at this point, I wanted to challenge myself by using new tools. This website is actually a Django Application that utilizes Bootstrap components along with my own CSS and JavaScript. It is a relatively simple application, but it shows just how clunky and unintuitive the current project archive system is.

While I doubt this application will ever reach production due to organizational contraints, I hope this demo gives you an idea of the power of the techonology. If you have any questions, feel free to contanct me at the email listed.
